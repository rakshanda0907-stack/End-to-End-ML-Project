# Employee Salary Prediction using Machine Learning

## Project Overview

This project demonstrates an **End-to-End Machine Learning Pipeline** for predicting employee salaries using **Linear Regression**. The workflow includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, prediction, and model serialization.
